# 🎰 Python Slot Machine Game

A console-based slot machine game built with Python that simulates a simple casino experience. Players can deposit money, place bets on multiple paylines, spin the reels, and win payouts based on matching symbols.

## ✨ Features

* 💰 Deposit and manage your balance
* 🎲 Randomly generated 3×3 slot machine
* 🎯 Bet on 1–3 paylines
* 💵 Adjustable betting limits
* 🏆 Symbol-based payout system
* 🎉 Jackpot bonus when all selected paylines win
* ✅ Input validation for deposits, bets, and paylines
* 🔄 Play multiple rounds until you choose to quit

## 🎮 How It Works

1. Deposit an initial balance.
2. Choose how many lines (1–3) you want to bet on.
3. Enter your bet amount for each line.
4. Spin the slot machine.
5. Matching symbols across a selected horizontal line earn a payout.
6. If every selected line is a winning line, you receive an additional jackpot bonus.
7. Your balance is updated after each spin.
